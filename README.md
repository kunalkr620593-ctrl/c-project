# 🎮 Number Guessing Game (C Language)

A simple and interactive **Number Guessing Game** built using the C programming language.
This project demonstrates the use of **loops, conditional statements, and random number generation**.

==🚀 Features

 🎯 Three difficulty levels:

  * Easy (1 – 75)
  * Medium (1 – 200)
  * Hard (1 – 500)

 🔢 Random number generation using `rand()` and `srand()`

📉 Smart hints:

  * "High"
  * "Low"

* ⭐ Performance rating system:

  * ⭐⭐⭐ Excellent
  * ⭐⭐ Good
  * ⭐ Average
  * 😅 Poor
## 🛠️ Technologies Used

* C Programming Language
* Standard Libraries:

  * `stdio.h`
  * `stdlib.h`
  * `time.h`

=== ▶️ How to Run==
 1. Compile the code

bash
gcc number gussing game.c 
==2. Run the program==

```bash
./game

==🎮 How to Play==
1. Select difficulty level:
   * 1 → Easy
   * 2 → Medium
   * 3 → Hard

2. Enter your guesses

3. Follow hints:

   * If guess is high → try lower number
   * If guess is low → try higher number

4. Keep guessing until you find the correct number 🎉
